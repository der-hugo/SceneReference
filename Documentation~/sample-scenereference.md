# About SceneReference Example

Example implementation showing the usage of `SceneReference`

## Dependencies

 - Uses `TextMeshPro`

## Installation

Install the samples via the PackageManager on the `derHugo - Scene Reference` package.

## Content

After installing the sample find the asset `Scene Reference Example` in the `Samples` folder.

It has a couple of scenes assigned for you to examine and test the drawer.

[![SceneReference Inspector][1]][1]

In the same folder you can also find the Scene named `SceneReferenceExample`. Open it and enter playmode.

Here you can see a list of buttons according to the scenes configured in the above mentioned asset.
You can load and unload the scenes async additive.

The scripts in this example are meant to serve as orientation for how to configure and consume `SceneReference` in your project.
